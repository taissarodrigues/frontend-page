## Projeto Landing Page Saint Seiya

Este é um projeto de landing page responsiva inspirado no universo de Saint Seiya, desenvolvido para apresentar os Cavaleiros de Atena, os Cavaleiros de Ouro e as 12 casas do zodíaco em uma experiência visual moderna, organizada e acessível.

O projeto foi construído utilizando HTML, CSS e JavaScript, com apoio de Bootstrap, Owl Carousel, Fancybox e jQuery para criar navegação responsiva, componentes interativos e uma apresentação fluida em desktop, tablet e mobile.

## Características

- Layout inspirado no universo de Saint Seiya
- Estrutura visual seguindo o layout proposto
- Menu responsivo com botão hamburger
- Navegação adaptada para tablet e mobile
- Hero com imagens de destaque e marcadores interativos
- Seção sobre os Cavaleiros de Atena
- Accordion com listas dos Cavaleiros de Ouro, Prata e Bronze
- Carousel interativo com os 12 Cavaleiros de Ouro
- Galeria das 12 casas com abertura em Fancybox
- Formulário de inscrição com validação em JavaScript
- Feedback visual de erro e sucesso nos campos do formulário
- Organização de classes seguindo boas práticas com BEM
- Medidas de tamanho estruturadas em REM

## Comportamento da Aplicação

- O menu se adapta automaticamente para telas menores usando o componente collapse do Bootstrap
- Os marcadores do hero alternam as imagens de destaque e atualizam o estado ativo
- O accordion organiza as categorias de Cavaleiros e permite expandir ou recolher cada lista
- O carousel exibe os Cavaleiros de Ouro com navegação customizada e comportamento responsivo
- A galeria das 12 casas abre as imagens em Fancybox para melhor visualização
- O formulário valida nome, e-mail, idade, altura, peso, data de nascimento, tipo sanguíneo, local de nascimento e local de treinamento
- Os campos do formulário exibem mensagens de erro acessíveis e estados visuais de validação
- O layout se ajusta para desktop, tablet e mobile mantendo consistência visual

## Objetivo do Projeto

Este projeto teve como objetivo praticar:

- Construção de landing page responsiva seguindo layout proposto
- Uso de HTML semântico, CSS modular e JavaScript puro
- Implementação de menu mobile com Bootstrap
- Uso de Carousel com jQuery e Owl Carousel 2
- Uso de Fancybox para galeria de imagens
- Implementação de Accordion com Bootstrap
- Aplicação do método BEM para organização das classes CSS
- Padronização de medidas em REM
- Validação de formulário com feedback visual e mensagens de erro
- Boas práticas de acessibilidade em navegação, imagens e formulário
